## URLs

relative URLs start with /
```markdown
[click here](/get-started/)
```

Use a trailing / for internal URLs but not if there is a #.
```markdown
[click here](/api-docs/)
[click here](/api-docs/#apiocl_system)
```

Use / to go to main page.
```markdown
[click here](/)
```
